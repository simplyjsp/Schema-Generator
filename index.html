Business Schema Generator

How to Use

  1. Select the schema types relevant to your business from the checkboxes below.
  2. Fill in the information for each selected schema type in the input fields that appear.
  3. Click the "Generate Schema" button to create your schema.
  4. Copy the generated schema or use the download buttons to save it as JSON or TXT.

Service Area Help

For the Service Area schema:

  • type: Use 'GeoCircle' for a circular area or 'GeoShape' for a specific region.
  • geoMidpoint: Enter as "latitude,longitude" (e.g., "40.7128,-74.0060" for New York City).
  • geoRadius: Enter the radius in kilometers (the "km" unit will be added automatically).
